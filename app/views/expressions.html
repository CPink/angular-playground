Angular helps bridge the gap between your html, css, and js

A cool feature of angular is the ability to use Expressions in our html

Example: 2+2 = {{ 2 + 2 }}

Exercises

  • Write a math equation that Angular can evaluate
    {{2+2}}
  • Using string concatination get the customary Hello, World! to display on the screen
    {{"Hello, " + "World!"}}
  • Use a currency filter to display the number 2.45 as $2.45
    {{2.45 | currency}}
  • Using a property called test from our ViewModel Controller render
    {{controller.test}}
